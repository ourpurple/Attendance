## 系统简介

考勤与请假管理系统是一个集考勤打卡、请假申请、加班申请、审批管理于一体的综合性管理系统。系统支持**移动端网页**和**微信小程序**两种访问方式，方便员工随时随地使用。

### 主要功能

- ✅ **考勤打卡**：上下班打卡，自动记录位置和时间
- ✅ **请假管理**：请假申请、查看、撤回
- ✅ **加班管理**：加班申请、查看、撤回
- ✅ **审批管理**：多级审批流程（部门主任、副总、总经理）
- ✅ **统计分析**：个人考勤统计、出勤记录查询

### 访问方式

1. **移动端网页**：在手机浏览器中访问系统地址   https://oa.ruoshui-edu.cn 
2. **微信小程序**：在微信中搜索并打开小程序

## 员工使用指南

### 一、考勤打卡

#### 1.1 上班打卡

**操作步骤：**

1. 打开系统首页
2. 查看当前时间和日期
3. 点击"**上班打卡**"按钮（蓝色按钮）
4. 系统会自动获取您的位置信息
5. 打卡成功后，按钮会变为灰色且不可点击

**注意事项：**

- ⏰ **打卡时间**：根据公司规定的上班时间（如：09:00）
- 📍 **位置要求**：需要允许浏览器/小程序获取位置权限
- ⚠️ **迟到提醒**：如果超过上班时间打卡，系统会弹出确认提示
- 🚫 **非工作日**：周末和法定节假日无法打卡（系统会自动隐藏打卡按钮）

#### 1.2 下班打卡

**操作步骤：**

1. 打开系统首页
2. 点击"**下班打卡**"按钮（绿色按钮）
3. 系统会自动获取您的位置信息
4. 打卡成功后，按钮会变为灰色且不可点击

**注意事项：**

- ⏰ **打卡时间**：根据公司规定的下班时间（如：18:00）
- ⚠️ **早退提醒**：如果早于下班时间打卡，系统会弹出确认提示
- ✅ **前提条件**：必须先完成上班打卡，才能进行下班打卡

#### 1.3 查看打卡记录

1. 在首页向下滑动，查看"**最近考勤**"区域
2. 点击"**考勤记录**"标签，查看详细记录
3. 可以按月份筛选查看历史记录

**记录信息包括：**

- 📅 打卡日期
- ⏰ 上班时间、下班时间
- ⚠️ 迟到/早退标记（如有）
- ⏱️ 工作时长

---

### 二、请假申请

#### 2.1 提交请假申请

**操作步骤：**

1. 在首页点击"**请假申请**"图标，或点击"**请假**"标签
2. 点击右上角"**+ 申请请假**"按钮
3. 填写请假信息：
   - **开始日期**：选择请假开始日期
   - **结束日期**：选择请假结束日期
   - **请假天数**：系统自动计算，也可手动输入（支持半天，如：0.5、1.5）
   - **请假原因**：详细说明请假事由（必填）
4. 点击"**提交**"按钮

**审批流程：**

根据请假天数，系统会自动分配审批人：

- **1天以内**：部门主任审批 → 完成
- **1-3天（含3天）**：部门主任审批 → 副总审批 → 完成
- **3天以上**：部门主任审批 → 副总审批 → 总经理审批 → 完成

#### 2.2 查看请假记录

1. 在"**请假**"标签页查看所有请假记录
2. 每条记录显示：
   - 📅 请假日期范围
   - 📊 请假天数
   - 📝 请假原因
   - ✅ 审批状态（待审批、已批准、已拒绝、已撤回）
   - 👤 待审批人（如果处于审批中）

**状态说明：**

- **待审批**：申请已提交，等待审批
- **已批准**：所有审批人已批准，请假生效
- **已拒绝**：审批被拒绝，请假无效
- **已撤回**：您主动撤回了申请

#### 2.3 撤回请假申请

**操作步骤：**

1. 在请假列表中，找到状态为"**待审批**"的记录
2. 点击"**撤回申请**"按钮
3. 确认撤回

**注意事项：**

- ✅ 只有"待审批"状态的申请可以撤回
- ❌ 已批准或已拒绝的申请无法撤回
- ⚠️ 撤回后无法恢复，如需请假需重新申请

---

### 三、加班申请

#### 3.1 提交加班申请

**操作步骤：**

1. 在首页点击"**加班申请**"图标，或点击"**加班**"标签
2. 点击右上角"**+ 申请加班**"按钮
3. 填写加班信息：
   - **开始时间**：选择加班开始日期和时间
   - **结束时间**：选择加班结束日期和时间
   - **加班天数**：系统自动计算，也可手动输入
     - 半天 = 0.5 天
     - 整天 = 1 天
     - 自定义 = x.5 天（如：1.5、2.5）
   - **加班原因**：详细说明加班事由（必填）
4. 点击"**提交**"按钮

**审批流程：**

- 员工加班申请 → 部门主任审批 → 完成

#### 3.2 查看加班记录

1. 在"**加班**"标签页查看所有加班记录
2. 每条记录显示：
   - 📅 加班时间范围
   - 📊 加班天数
   - 📝 加班原因
   - ✅ 审批状态（待审批、已批准、已拒绝、已撤回）
   - 👤 待审批人（如果处于审批中）

#### 3.3 撤回加班申请

**操作步骤：**

1. 在加班列表中，找到状态为"**待审批**"的记录
2. 点击"**撤回申请**"按钮
3. 确认撤回

---

### 四、统计分析

#### 4.1 查看个人统计

1. 在首页点击"**统计分析**"图标，或点击"**统计**"标签
2. 选择要查看的月份
3. 查看统计数据：

**统计内容：**

- 📊 **出勤天数**：实际出勤的工作日数
- ⏱️ **工作时长**：总工作小时数
- ⚠️ **迟到次数**：迟到打卡的次数
- ⚠️ **早退次数**：早退打卡的次数
- 🏖️ **请假天数**：当月请假总天数
- ⏰ **加班天数**：当月加班总天数

## 部门主任使用指南

部门主任除了拥有**员工的所有功能**外，还拥有**审批权限**。以下重点说明审批功能。

### 一、审批功能入口

1. 登录系统后，在首页可以看到"**我的审批**"图标（如果有待审批事项，会显示红色数字徽章）
2. 点击"**我的审批**"图标，进入审批页面
3. 或直接点击底部"**审批**"标签

### 二、审批请假申请

#### 2.1 查看待审批请假

1. 进入"**我的审批**"页面
2. 默认显示"**请假**"标签页
3. 查看待审批的请假申请列表

**列表信息：**

- 👤 **申请人**：申请人的姓名
- 📅 **请假日期**：请假的起止日期
- 📊 **请假天数**：请假的总天数
- 📝 **请假原因**：申请人的请假说明
- ✅ **审批状态**：当前审批状态

#### 2.2 审批操作

**批准申请：**

1. 点击要审批的申请，查看详情
2. 点击"**批准**"按钮
3. （可选）输入批准意见
4. 确认批准

**拒绝申请：**

1. 点击要审批的申请，查看详情
2. 点击"**拒绝**"按钮
3. **必须输入拒绝理由**（必填）
4. 确认拒绝

**注意事项：**

- ✅ 部门主任只能审批**本部门员工**的请假申请
- ⏰ 审批后，系统会自动流转到下一级审批人（如需要）
- 📝 审批意见会记录在系统中，申请人可以查看

### 三、审批加班申请

#### 3.1 查看待审批加班

1. 在"**我的审批**"页面，点击"**加班**"标签
2. 查看待审批的加班申请列表

#### 3.2 审批操作

操作方式与审批请假相同：

- **批准**：点击"批准"按钮，可输入批准意见
- **拒绝**：点击"拒绝"按钮，必须输入拒绝理由

**注意事项：**

- ✅ 部门主任只能审批**本部门员工**的加班申请
- 📝 审批意见会记录在系统中

### 四、查看本部门统计

部门主任可以查看本部门的考勤统计（如果系统管理员已授权）：

1. 进入"**统计分析**"页面
2. 选择要查看的月份或日期范围
3. 查看本部门员工的统计信息

## 副总使用指南

副总除了拥有**员工的所有功能**外，还拥有**审批权限**。副总的审批权限范围更广，可以审批多个部门的申请。

### 一、审批功能入口

1. 登录系统后，在首页可以看到"**我的审批**"图标（如果有待审批事项，会显示红色数字徽章）
2. 点击"**我的审批**"图标，进入审批页面

### 二、审批请假申请

#### 2.1 查看待审批请假

1. 进入"**我的审批**"页面
2. 查看"**请假**"标签页中的待审批申请

**审批范围：**

- ✅ 可以审批分配给自己的请假申请
- ✅ 可以审批分管部门的员工申请（根据系统配置的分管关系）
- ✅ 可以审批自己的请假申请（如果申请时选择了"本人审批"）

#### 2.2 审批操作

操作方式与部门主任相同：

- **批准**：点击"批准"按钮，可输入批准意见
- **拒绝**：点击"拒绝"按钮，必须输入拒绝理由

**审批流程：**

- 如果请假天数 ≤ 3天：副总审批后，申请完成
- 如果请假天数 > 3天：副总审批后，流转到总经理审批

### 三、审批加班申请

#### 3.1 查看待审批加班

1. 在"**我的审批**"页面，点击"**加班**"标签
2. 查看待审批的加班申请

**审批范围：**

- ✅ 可以审批分配给自己的加班申请
- ✅ 可以审批分管部门的员工申请
- ✅ 可以审批自己的加班申请（如果申请时选择了"本人审批"）

#### 3.2 审批操作

操作方式与部门主任相同。

### 四、提交请假申请（特殊说明）

副总提交请假申请时，可以**手动指定审批人**：

1. 在请假申请页面，填写基本信息
2. 如果请假天数 ≤ 3天：
   - 可以指定副总审批人（默认是"本人审批"，可选择其他副总）
3. 如果请假天数 > 3天：
   - 可以指定副总审批人（默认是"本人审批"）
   - 可以指定总经理审批人（系统自动分配或手动选择）

**审批流程：**

- **3天以内（含3天）**：副总审批（默认本人，可指定其他副总）→ 完成
- **3天以上**：副总审批（默认本人，可指定其他副总）→ 总经理审批 → 完成

### 五、提交加班申请（特殊说明）

副总提交加班申请时，可以**手动指定审批人**：

1. 在加班申请页面，填写基本信息
2. 可以指定审批人（默认是"本人审批"，可选择其他副总）

**审批流程：**

- 副总审批（默认本人，可指定其他副总）→ 完成

## 总经理使用指南

总经理拥有**最高审批权限**，可以审批所有需要总经理审批的申请。

### 一、审批功能入口

### 二、审批请假申请

#### 2.1 查看待审批请假

**审批范围：**

操作方式与其他审批人相同。

### 五、提交加班申请（特殊说明）

总经理提交加班申请时：

1. 在加班申请页面，填写基本信息
2. 系统默认由**总经理本人直接审批**
3. 申请提交后，立即完成审批（无需其他人审批）

### 六、查看全公司统计

总经理可以查看全公司的考勤统计（如果系统管理员已授权）：

1. 进入"**统计分析**"页面
2. 选择要查看的月份或日期范围
3. 查看所有员工的统计信息

## 系统管理员使用指南

系统管理员拥有系统的**最高管理权限**，可以通过**管理后台**进行全面的系统管理和数据查看。管理后台只能在电脑浏览器中访问。

### 一、登录管理后台

#### 1.1 访问地址

在电脑浏览器中访问管理后台地址（如：`http://oa.ruoshui-edu.cn/admin`）

**注意：** 只有**系统管理员**账号才能登录管理后台，其他角色无法登录。

#### 1.2 登录步骤

1. 打开管理后台登录页面
2. 输入**管理员用户名**和**密码**
3. 点击"登录"按钮

**默认管理员账号：**

- 用户名：`admin`
- 密码：`admin123`

**⚠️ 重要：首次登录后请立即修改密码！**

---

### 二、仪表盘

登录后首先看到的是**仪表盘**，显示系统关键数据概览：

#### 2.1 数据卡片

- **总用户数**：系统中所有用户的总数
- **今日出勤**：今天已打卡的员工数量
- **待审批请假**：当前待审批的请假申请数量
- **待审批加班**：当前待审批的加班申请数量

#### 2.2 功能说明

仪表盘提供系统运行状态的快速概览，帮助管理员了解：
- 系统用户规模
- 当日考勤情况
- 待处理审批事项

---

### 三、用户管理

#### 3.1 查看用户列表

1. 点击左侧菜单"**用户管理**"
2. 查看所有用户列表，包括：
   - 用户ID、用户名、姓名
   - 手机号
   - 角色（员工、部门主任、副总、总经理、管理员）
   - 所属部门
   - 账户状态（激活/禁用）

#### 3.2 添加用户

**操作步骤：**

1. 点击"**添加用户**"按钮
2. 填写用户信息：
   - **基本信息**：
     - 用户名（必填，用于登录）
     - 姓名（必填）
     - 手机号（可选）
     - 邮箱（可选）
   - **权限设置**：
     - 角色（必选）：员工、部门主任、副总、总经理、管理员
     - 所属部门（根据角色选择，副总/总经理可不选）
     - 账户状态：激活/禁用
   - **密码设置**：
     - 初始密码（必填）
3. 点击"**确定**"保存

**注意事项：**

- 用户名必须唯一
- 副总、总经理可以不设置部门
- 建议设置强密码，并提醒用户首次登录后修改

#### 3.3 编辑用户

**操作步骤：**

1. 在用户列表中找到要编辑的用户
2. 点击"**编辑**"按钮
3. 修改用户信息（可修改所有字段，包括角色、部门、状态）
4. 点击"**确定**"保存

**注意事项：**

- 修改角色后，用户权限会立即生效

## 常见问题

### Q1: 忘记密码怎么办？

**答：** 请联系系统管理员重置密码。系统管理员可以在管理后台重置您的密码。

### Q2: 打卡时提示"无法获取位置"？

**答：** 
1. 检查浏览器/小程序的定位权限是否已开启
2. 确保手机GPS功能已开启
3. 尝试在室外或信号较好的地方打卡
4. 刷新页面后重试

### Q3: 为什么周末无法打卡？

**答：** 系统会自动识别周末和法定节假日，这些日期无需打卡。如果周末需要上班（调休工作日），系统管理员会提前配置，届时可以正常打卡。

### Q4: 请假申请提交后可以修改吗？

**答：** 不可以直接修改。如果申请还在"待审批"状态，您可以**撤回申请**，然后重新提交。如果已经审批通过或拒绝，则无法撤回。

### Q5: 如何查看审批进度？

**答：** 在请假/加班列表中，点击申请记录可以查看详情，包括：
- 当前审批状态
- 待审批人姓名
- 已审批人的意见和时间

### Q6: 审批时可以不填写意见吗？

**答：** 
- **批准**：可以不填写意见（可选）
- **拒绝**：必须填写拒绝理由（必填）

### Q7: 为什么看不到"我的审批"功能？

**答：** "我的审批"功能只对**部门主任、副总、总经理**开放。普通员工没有审批权限，因此看不到此功能。

### Q8: 打卡后按钮还是可以点击？

**答：** 打卡成功后，页面会自动刷新。如果按钮仍然可点击，请：
1. 手动刷新页面
2. 检查网络连接
3. 查看打卡记录确认是否已成功

### Q9: 如何查看历史考勤记录？

**答：** 
1. 在首页点击"**考勤记录**"标签
2. 选择要查看的月份
3. 查看该月的所有打卡记录

### Q10: 请假天数如何计算？

**答：** 
- 系统会根据开始日期和结束日期自动计算
- 支持半天请假（如：0.5天、1.5天）
- 可以手动调整天数

### Q11: 副总如何指定其他副总审批？

**答：** 
1. 在提交请假/加班申请时
2. 在"指定审批人"下拉菜单中选择其他副总
3. 如果不选择，默认是"本人审批"

### Q12: 系统支持哪些浏览器？

**答：** 
- **移动端网页**：推荐使用 Chrome、Safari、微信内置浏览器
- **微信小程序**：在微信中直接使用

### Q13: 数据会丢失吗？

**答：** 不会。所有数据都保存在服务器数据库中，包括：
- 打卡记录
- 请假/加班申请
- 审批记录

即使更换设备或重新登录，数据都会保留。

### Q14: 如何联系技术支持？

**答：** 如遇到技术问题，请联系系统管理员或IT支持人员。

## 附录：审批流程图

### 请假审批流程

```
员工/部门主任请假：
├─ 1天以内
│  └─ 部门主任审批 → 完成
│
├─ 1-3天（含3天）
│  └─ 部门主任审批 → 副总审批 → 完成
│
└─ 3天以上
   └─ 部门主任审批 → 副总审批 → 总经理审批 → 完成

副总请假：
├─ 3天以内（含3天）
│  └─ 副总审批（默认本人，可指定其他副总）→ 完成
│
└─ 3天以上
   └─ 副总审批（默认本人，可指定其他副总）→ 总经理审批 → 完成

总经理请假：
└─ 总经理本人审批 → 完成
```

### 加班审批流程

```
员工/部门主任加班：
└─ 部门主任审批 → 完成

副总加班：
└─ 副总审批（默认本人，可指定其他副总）→ 完成

总经理加班：
└─ 总经理本人审批 → 完成
```
